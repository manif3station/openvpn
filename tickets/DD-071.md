## Title

## Acceptance

- `dashboard openvpn.setup` stores username, password, and optional 2FA data in `~/.openvpn.env`
- missing setup arguments prompt for only the missing values
- `dashboard openvpn.connect --collector` returns `OVPN?` state before setup is complete
- collector mode reconnects automatically when setup exists and auto reconnect is enabled
- five failed reconnect attempts disable reconnect until the user runs `dashboard openvpn.connect --auto`
- `dashboard openvpn.disconnect` disables reconnect and disconnects the managed process
- `dashboard openvpn.noreconnect` disables reconnect without disconnecting
- Docker tests and 100% coverage pass
- docs, commit, and push are completed
